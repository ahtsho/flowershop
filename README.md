-------------------------------------------------------------------------------

Please list your orders in the following format:
10 R12
15 L09
13 T58
and place them by pressing Enter twice.

To tun the progam with sample input pass it the argument "-t"

>> java com.icare.flowershop.ConsoleFlowershopRunner -t

10 R12 $12.99

15 L09 $41.90
   1 x 9 $24.95
   1 x 6 $16.95
13 T58 $25.85
   2 x 5 $9.95
   1 x 3 $5.95
